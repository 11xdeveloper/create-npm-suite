## Usage

Run the create command of your favourite package manager!

e.g.

```bash
bun create npm-suite@latest
```

OR

```bash
pnpm create npm-suite@latest
```

OR

```bash
npm create npm-suite@latest
```

OR

```bash
yarn create npm-suite@latest
```
